# True Aim

Visual feedback overlay for input directions in Hollow Knight: Silksong.

## Features

Press **F8** (keyboard) or **L3/Left Stick Click** (controller) to toggle the directional overlay that shows where your attacks will go.

## Installation

Install via r2modman or Thunderstore Mod Manager. BepInEx will be installed automatically as a dependency.

Manual installation: Extract to your game folder and merge with existing BepInEx installation.

## Configuration

Press F8/L3 in-game to toggle the overlay. Additional settings available in BepInEx config file.

## Dependencies

BepInEx with configuration manager (required for customization/localization features).

# 真准 - 中文简介

输入方向可视化反馈模组。

按下 **F8**（键盘）或 **L3**（左摇杆按键）切换显示攻击方向的覆盖层。
